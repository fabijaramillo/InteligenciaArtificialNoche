Ciberseguridad, Inteligencia Artificial y Computación en la Nube

En el webinar sobre Ciberseguridad, Inteligencia Artificial y Computación en la Nube se aprendió que la ciberseguridad se utiliza para prevenir ataques de malware y phishing, los cuales son amenazas que atacan sin el consentimiento de las personas con el fin de robar información.

Se mencionó que se emplea mucho la IA y la computación en la nube para reducir costos de administración y agilizar la gestión de procesos, tales como transacciones, retiros y transferencias de dinero. De esta manera, se propone el uso adecuado de las nuevas tecnologías para garantizar la seguridad de las personas y cómo estas manejan su dinero.

Cuando otorgamos permisos a páginas o plataformas, se debe tener en cuenta si realmente deseamos dar acceso al tratamiento de nuestros datos personales. Esta regulación está definida en la Ley de Protección de Datos Personales o Ley 1581 de 2012, la cual reconoce y protege el derecho que tienen todas las personas a conocer, actualizar y rectificar la información que se haya recogido sobre ellas. Esta información queda guardada en bases de datos o archivos que son susceptibles de tratamiento por entidades de naturaleza pública o privada.

¿Qué es la Inteligencia Artificial?

La inteligencia artificial (IA) es un campo de la informática que se centra en desarrollar sistemas y tecnologías capaces de realizar tareas que normalmente requieren inteligencia humana. Esto incluye la capacidad de aprender de experiencias, adaptarse a nuevas situaciones, comprender y procesar el lenguaje natural, reconocer patrones visuales y realizar tareas complejas. La IA se utiliza en aplicaciones como asistentes virtuales, sistemas de recomendación, vehículos autónomos y muchas otras áreas, y su objetivo es emular o superar la capacidad cognitiva humana en una variedad de contextos.

Resumen del Código punto 4

  1. Importaciones de bibliotecas:
    import tensorflow as tf
    import numpy as np
    Se importan las bibliotecas TensorFlow y NumPy. TensorFlow es popular para el aprendizaje automático y la creación de redes neuronales, mientras que NumPy se utiliza para trabajar con arreglos y cálculos numéricos.
  2. Creación de arreglos:
    celsius = np.array([-40, -10, 0, 8, 15, 22, 38], dtype=float)
    fahrenheit = np.array([-40, 14, 32, 59, 72, 100], dtype=float)
    Se crean dos arreglos de NumPy: celsius (temperaturas en Celsius) y fahrenheit (temperaturas en Fahrenheit). El parámetro dtype=float asegura que los elementos sean flotantes.
  3. Definición de una capa de la red neuronal:
    capa = tf.keras.layers.Dense(units=1, input_shape=[1])
    Se define una capa densa con una sola neurona de salida. input_shape=[1] especifica que la entrada será un solo valor.
  4. Creación del modelo:
    modelo = tf.keras.Sequential([capa])
    Se crea un modelo secuencial que contiene la capa definida anteriormente.
  5. Compilación del modelo:
    modelo.compile(optimizer=tf.keras.optimizers.Adam(0.1), loss='mean_squared_error')
    Se compila el modelo con el optimizador Adam y se establece la función de pérdida como error cuadrático medio (MSE).
  6. Inicio del entrenamiento:
    print("Comenzando entrenamiento...")
    Se imprime un mensaje indicando que el entrenamiento está a punto de comenzar.
  7. Entrenamiento del modelo:
    historial = modelo.fit(celsius, fahrenheit, epochs=1000, verbose=False)
    El modelo se entrena durante 1000 épocas con los datos de celsius y fahrenheit.
  8. Finalización del entrenamiento:
    print("Modelo entrenado!")
    Se imprime un mensaje indicando que el modelo ha terminado de entrenarse.
  9. Predicción:
    resultado = modelo.predict([100.0])
    Se realiza una predicción para 100 grados Celsius, y el resultado se almacena en la variable resultado.
  10. Impresión del resultado:
    print("El resultado es " + str(resultado) + " fahrenheit!")
    Se imprime el resultado de la predicción.

En resumen, este fragmento de código establece un modelo de red neuronal simple que predice temperaturas en Fahrenheit a partir de temperaturas en Celsius y muestra el resultado en la consola.
